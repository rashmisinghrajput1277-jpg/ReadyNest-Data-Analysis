# ReadyNest-Real State-Analysis
  Built an end-to-end data analytics project for ReadyNest during my internship by cleaning real estate transaction data using Python/Pandas and creating an interactive Tableau dashboard to analyze profit by property segment and regional sales. Key insight: Family Modular led profitability, with 1,162 total units sold across regions.
  Core Business Takeaways 
Top-Performing Categories: A deep-dive analysis shows that a small, specific segment of your product line is driving the vast majority of overall sales revenue, meaning marketing spend should be focused heavily there.

Seasonal Sales Spikes: Time-series patterns reveal a consistent, predictable surge in transactions during specific periods, allowing the team to better plan inventory and server scaling ahead of time.

Customer Retention Gap: While initial user acquisition numbers look strong, there is a sharp drop-off after the first purchase, showing a clear need for a post-purchase follow-up campaign to bring customers back.
